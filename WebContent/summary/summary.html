HTML SUMMARY

1. HTML & CSS & JavaScript

1) HTML : 웹 페이지의 내용을 작성

- HyperText : 웹 페이지의 특정 부분과 연결이 가능한 텍스트
- Mark-up Language : 정보를 구조적, 계층적으로 표현할 수 있는 언어

2) CSS : 웹 페이지의 모양 및 스타일을 지정

3) HTML : 웹 페이지의 기능 및 동작을 지정


2. HTML elements

1) HTML 요소 : 태그

2) 브라우저 : HTML 문서 상의 태그를 해석하여, 그 내용을 표시하는 프로그램

3) 구성

<시작태그> content <끝태그>

- 끝태그가 없는 '빈 태그'도 존재하며, 특수한 용도로 사용됨
